# AWS Infrastructure Builder

Ferramenta educacional para criação e validação de diagramas de arquitetura AWS.

> ⚠️ Este projeto foi desenvolvido exclusivamente para fins de estudo e prática. Não deve ser considerado uma ferramenta oficial da AWS.

## Objetivo

O principal objetivo deste projeto é reforçar e consolidar conhecimentos em:

- AWS Cloud Computing
- Arquitetura de soluções
- Boas práticas de infraestrutura
- Serviços AWS
- Relacionamentos entre recursos
- Well-Architected Framework
- Modelagem de ambientes em nuvem

O desenvolvimento deste projeto faz parte da minha jornada de aprendizado através da **Escola da Nuvem**, onde estou aprofundando meus conhecimentos em computação em nuvem e arquitetura AWS.

---

## Funcionalidades

- Criação visual de diagramas AWS
- Catálogo de serviços AWS
- Conexão entre recursos
- Validação de arquitetura
- Identificação de dependências ausentes
- Detecção de conexões inválidas
- Exportação de diagramas
- Geração de laboratórios (LAB Generator)
- Geração básica de comandos AWS CLI
- Geração básica de código Terraform

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- SVG

## Aviso

Este projeto possui caráter exclusivamente educacional.

Seu propósito é auxiliar na fixação de conceitos aprendidos durante os estudos de computação em nuvem, servindo como laboratório pessoal para experimentação de arquiteturas AWS.
